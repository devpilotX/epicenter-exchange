Algo signal demos for learning.

Classic technical strategies in clean Python — read the code, run the backtests, see what works and what doesn't.

No strategy is 100% accurate. Anyone claiming “guaranteed” signals is misleading you (and likely breaking SEBI/SEC/FCA rules). These demos exist to teach how strategies work, including their failure modes. Do not trade real money based on them.

Simple signals

1

SMA Crossover (50/200)

The classic trend-following “golden cross” / “death cross.” Works on indices, fails on choppy stocks.

python simple_signals.py --strategy sma

2

RSI Mean-Reversion

Buy when RSI < 30, sell when RSI > 70. Decent on range-bound equity; bad in strong trends.

python simple_signals.py --strategy rsi

3

MACD Histogram

12/26 EMA difference with 9-period signal line. Trend-confirmation tool, not a holy grail.

python simple_signals.py --strategy macd

Advanced signals

A

Bollinger + Z-Score Mean Reversion

Trade extremes of the price-to-mean z-score with a 20-day window. Good intuition builder.

B

12-1 Cross-Sectional Momentum

Rank a basket on trailing 12-month minus 1-month return; long top decile, short bottom. Classic academic factor.

C

ATR-Based Position Sizing

Size positions so each one risks the same fraction of capital, using ATR for volatility.

Where to find the code

Everything lives in the /algo folder on GitHub. Clone, install requirements, and run.

git clone https://github.com/devpilotX/epicenter-exchange.git
cd epicenter-exchange/algo
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python simple_signals.py --ticker RELIANCE.NS --strategy sma
python backtest.py --ticker BTC-USD --strategy rsi
Risk reminder: Backtests are historical simulations. They suffer from look-ahead bias, survivorship bias, transaction-cost approximations, and regime change. Real trading involves slippage, latency, taxes, and emotional pressure that backtests cannot capture.
