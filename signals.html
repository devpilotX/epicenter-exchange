Signals — open-source & educational

Algos you can read.

Textbook strategies in Python, MIT-licensed, with an honest backtester. The point is to understand how signals are constructed — not to trade them.

View code on GitHub
No accuracy claims, ever. No algorithm has 100% (let alone 1000%) accuracy. Backtests carry look-ahead and survivorship bias unless you actively prevent them. Real markets add slippage, taxes, latency, and emotional pressure. This code is for learning, not trading.

Simple signals

SMA

SMA Crossover (50/200)

The classic golden / death cross. Long when fast SMA > slow SMA, flat otherwise.

RSI

RSI(14) Mean-reversion

Buy bias below 30, sell bias above 70. Works best on range-bound assets, fails on trends.

MACD

MACD(12/26/9)

Histogram zero-crossings. A smoother variant of crossover with a momentum component.

Advanced signals

Z

Z-score mean-reversion

Trade extremes of a 20-day price-to-mean z-score. Long at −2σ, exit near the mean.

M

12-1 trend momentum

12-month return minus the most recent month. Classic factor used in many quant shops.

A

ATR position sizing

Size positions so one ATR of price movement equals a fixed % of capital at risk.

Honest backtester

The included backtest.py runs any signal long-only and reports total return, CAGR, annualised volatility, Sharpe ratio, max drawdown, and win rate. Signals computed on day t are lagged to day t+1 to avoid look-ahead. Costs and slippage are not modelled — add them yourself before drawing conclusions.

Quick start

git clone https://github.com/devpilotX/epicenter-exchange.git
cd epicenter-exchange/algo
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# Today's signal on a single ticker
python simple_signals.py --ticker RELIANCE.NS --strategy sma
python advanced_signals.py --ticker ^NSEI --strategy meanrev

# Full backtest with equity-curve plot
python backtest.py --ticker AAPL --strategy macd --start 2005-01-01
