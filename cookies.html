Cookie Policy

Last updated: 25 May 2026

One line: Epicenter Exchange does not set any tracking, analytics, or advertising cookies. There is no cookie banner because there is nothing to consent to.

1. What is a cookie?

A cookie is a small text file a website stores in your browser. Sites use them for things like logins, preferences, and tracking. We don't use them at all.

2. What we use instead

  • localStorage: the calculators on Tools remember your last-used inputs so a refresh doesn't wipe them. This data stays in your browser and is never sent anywhere. You can clear it any time from your browser's site-data settings.

3. Third-party cookies

Embedded third-party services may set their own cookies only when you actively interact with them:

  • Formspree may set a session cookie on the contact form submission. See Formspree's cookie policy.
  • Buttondown may set a session cookie on the newsletter subscription. See Buttondown's cookie policy.
  • CoinGecko API requests are pure JSON fetches and do not set cookies in your browser.

If you do not interact with these, no third-party cookies are set on your visit.

4. How to clear site data

In Chrome / Edge: ⋮ → Settings → Privacy and security → Site Settings → View permissions and data stored across sites → search “epicenterexchange” → Clear data. Similar paths exist in Firefox and Safari.

5. Changes

If we ever add analytics or any consent-required tracker, this page will be updated and a banner will appear on first visit. That's a hard promise.
